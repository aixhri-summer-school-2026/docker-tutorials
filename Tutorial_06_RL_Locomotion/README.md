# Tutorial 06 : PPO Locomotion

> It is **recommended** to use the `run_all_setup.sh` script (located in the root of `docker-tutorials`) to download the entire codebase. If you prefer to download only this tutorial individually, you can instead run the `setup_tutorial_??.sh` script provided inside the tutorial folder.

Make sure you have cloned this repository and are currently inside the correct directory:
```bash
cd docker-tutorials/Tutorial_06_RL_Locomotion
```

Make the setup script executable:
```bash
chmod +x setup_tutorial_06.sh
```

Run the setup script and specify the destination directory where the tutorial will be installed:
```bash
./setup_tutorial_06.sh ~/aixhri-summer-school/Tutorial_06_RL_Locomotion
```
The script will automatically create the target directory if it does not exist and set up all required files.
